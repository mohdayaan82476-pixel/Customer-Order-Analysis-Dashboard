# 📊 Customer & Order Analysis Dashboard

## 🧠 Project Overview

This project presents a **Customer & Order Analysis Dashboard** built using SQL and Power BI. It focuses on analyzing customer distribution and order trends to uncover key business insights across regions and cities.

---

## 🎯 Objectives

* Analyze customer distribution by state and city
* Identify regions with the highest number of orders
* Track order trends over time
* Understand geographic demand patterns

---

## 📁 Dataset

The dataset is based on an e-commerce database containing:

* Customer information (city, state)
* Order details (order ID, purchase timestamp)

---

## ⚙️ Tools & Technologies

* **SQL** – Data extraction and transformation (JOIN, GROUP BY)
* **Power BI** – Dashboard creation and visualization
* **Excel** – Data export and intermediate handling

---

## 📊 Key Features

### 🔹 KPI Cards

* Total Orders
* Total Customers

### 🔹 Visualizations

* 📈 Monthly Order Trend
* 🌍 Orders by State
* 🏙 Top Cities by Orders
* 👥 Customer Distribution by State

---

## 💡 Key Insights

* Certain states contribute significantly to total orders
* A few major cities dominate order volume
* Orders show variation across months
* Customer base is concentrated in specific regions

---

## 🔗 SQL Concepts Used

* INNER JOIN
* GROUP BY
* COUNT() aggregation
* Date-based analysis
